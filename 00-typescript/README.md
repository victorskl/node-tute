- Quick start
```
yarn build
node primitive
```

- Boostrap TypeScript Project
```
npm init

npm install typescript --save-dev
(OR)
yarn add typescript --dev

node_modules/typescript/bin/tsc --version

npx tsc --version
npx tsc --help
npx tsc --init
npx tsc greeter.ts
npx tsc
```
